# News App (Flutter)
A simple and clean News App built with Flutter. The app fetches news using NewsAPI, displays articles by category and source, and includes a working search feature. The UI is modern, responsive, and smooth.
## Features
- Latest news
- News by source
- Search articles
- Open full article in browser
- Fast image loading with CachedNetworkImage
- Modern and clean UI
- Responsive design using ScreenUtil
## API Used
The app uses NewsAPI: https://newsapi.org/  
Endpoints used: /v2/top-headlines/sources and /v2/everything
## How to Run
1. Clone the project:  
git clone <your-repo-url>
2. Install dependencies:  
flutter pub get
3. Add your API key inside:  
lib/core/resources/AppConstants.dart
4. Run the app:  
flutter run
